# 主页与统一总览页设计方案（餐饮/中资企业/供应商）

## 目标
- 首页在单页内快速预览餐饮、中资企业、供应商三块内容，提升信息查找效率。
- 新增统一总览页 `/directory`，可按 Tab 切换板块，并支持城市与分类筛选。
- 首页保持卡片方格风格，预览卡直达详情页，“查看更多”进入列表页或总览页。

## 首页信息架构
- 顶部保留小型标题区：**墨西哥华人商家与中资企业导航**。
- 标题下方放全站城市筛选条（胶囊样式，横向可滚动）：`全部 / 墨西哥城 / 蒙特雷 / 瓜达拉哈拉`，显示数量。
- 主体为左右两列：
  - 左列：餐饮板块 + 供应商板块（上下两块，互相独立）。
  - 右列：中资企业板块。
- 每个板块 3 个格子：2 个预览卡 + 1 个“查看更多”卡。
  - 预览卡：封面图 + 标题 + 简短描述，点击进入 `/company/:slug`。
  - 查看更多：跳转到 `/directory?tab=restaurants` 或 `/directory?tab=enterprises`。
  - 供应商板块：静态占位卡（“即将上线/征集中”） + CTA。

## 数据与规则
- 数据来源：`data/companies.json`。
- 餐饮板块：`industry = 餐饮与服务`。
- 中资企业板块：`industry = 中资企业`。
- 预览卡取“当前筛选列表的最后两条”。
- 城市筛选默认“全部”，用户选择后使用 `localStorage` 记忆。

## 统一总览页 `/directory`
- 顶部结构：小标题区 + 城市筛选条 + Tab（餐饮 / 中资企业 / 供应商）。
- 城市筛选全局生效；Tab 切换不重置已选城市。
- 餐饮 Tab 展示分类筛选（中餐/超市/火锅/面馆/饮品/烧烤）。
- 企业与供应商 Tab 不显示分类筛选。
- 列表展示为卡片网格，风格与首页一致，点击卡片进入详情页。
- URL 参数支持：`/directory?tab=restaurants&city=all`，用于首页“查看更多”直达。

## 视觉与交互
- 统一使用现有卡片样式，但在首页增加更精致的板块标题/数量胶囊。
- 城市筛选条采用可滚动胶囊，贴近参考图样式。
- 切换城市与 Tab 后，数量与内容实时更新。
